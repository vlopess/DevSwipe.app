# DevSwipe

**DevSwipe** is a modern web app to discover tech jobs in a fast, swipe-based experience.

Think **Tinder**, but for developers 💻🔥

Built with **vibe coding** — fast, experimental, and focused on solving a real problem.

## 🚀 How it works

* Browse tech jobs in a swipeable interface
* Swipe right to like a job
* Swipe left to skip
* Open details instantly
* Apply directly from the platform

---

## ⚡ Features

* Swipe-based job discovery
* Aggregation of multiple job APIs
* Smart filtering (role, stack, keywords)
* Real-time job fetching
* Clean and responsive UI
* Optimized for speed and simplicity
